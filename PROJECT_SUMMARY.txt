ğŸ”Œ API ENDPOINTS
================================================================================

AUTHENTICATION
  POST   /api/auth/signup              Register new user
  POST   /api/auth/login               Login user
  GET    /api/auth/me                  Get current user (protected)

EVENTS
  GET    /api/events                   Get all events (with pagination)
  GET    /api/events/:id               Get event details
  GET    /api/events/search?q=query    Search events
  POST   /api/events                   Create event (organizer only)
  PUT    /api/events/:id               Update event (owner only)
  DELETE /api/events/:id               Delete event (owner only)

REGISTRATIONS
  POST   /api/registrations/:eventId   Register for event
  DELETE /api/registrations/:eventId   Unregister from event
  GET    /api/registrations/my         Get my registrations
  GET    /api/registrations/event/:id  Get event attendees (owner only)

FEEDBACK
  POST   /api/feedback                 Submit feedback
  GET    /api/feedback                 Get all feedback (admin only)
  PUT    /api/feedback/:id             Update feedback status (admin only)
